# BLACKSTONE - Responsive Company Website

## Overview

BLACKSTONE is a modern, responsive multi-page company website built using HTML, CSS, and JavaScript. The website showcases a professional business presence with a clean black-and-white design, responsive navigation, service highlights, company information, and a contact form.

---

## Features

- Fully Responsive Design
- Multi-Page Layout
- Sticky Navigation Bar
- Mobile Hamburger Menu
- Hero Section with Background Image
- About Us Page
- Services Showcase
- Contact Form with Validation
- Professional Footer
- Smooth User Experience
- Modern Black & White Theme

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
